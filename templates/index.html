DataSnap Help Guide

Welcome to DataSnap - your powerful data analysis tool. Here's how to make the most of it:

Getting Started

Drag and drop your data file into the upload area or click to browse. DataSnap supports CSV, Excel, and JSON formats.

Using Groups

Organize your analyses by assigning files to groups. Each group appears as a tab in the sidebar that you can:

  • Drag to undock as a separate analysis card
  • Right-click to rename
  • Hover to reveal the delete button

Workspace Navigation

  • Pan: Use the slider at the bottom to move left/right
  • Zoom: Hover over the magnifying glass and scroll
  • Drag: Move any card by its handle

Analysis Features

DataSnap automatically provides:

  • File metadata (size, rows, columns)
  • Data quality metrics (missing values, duplicates)
  • Column names and data preview
